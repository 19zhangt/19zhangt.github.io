Windows10 上利用Hexo搭建个人博客需要注意的事项

  • 发布 2018-09-29
  • 更新 2018-10-01
  • 阅读

  1. 在Github中的New repository创建==yourid==. github.io时，yourid必须和用户名一致，否则后面会加载不上。
  2. 使用npm安装node.js 包hexo-cli前记得更换源。
    可以使用使用淘宝镜像：临时 npm –registry https://registry.npm.taobao.org install express 永久 npm config set registry https://registry.npm.taobao.org
    npm没安装成功，记得npm cache clean –force 在进行重新安装。
  3. hexo博客可以建在电脑的任意盘，但是路径要对。如 hexo init D:\Blog。另外输入hexo new file；hexo g；hexo s以及最后的hexo d都需要进入建博客的目录中，如上面的D:\Blog。
  4. 更换主题，可选择的很多Hexo themes。需要考虑个人风格。我选择的是Yearn，嵌套了七牛云图床，DaoVoice等功能，风格简约。
    具体搭建流程参照了这个网站
