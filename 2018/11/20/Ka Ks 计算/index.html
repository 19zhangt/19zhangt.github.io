Ka Ks 计算

Ka/Ks 计算需要用到：

  1. ParaAT.pl(https://github.com/wonaya/ParaAT) 转换数据格式
  2. cat 合并输出的每一对结果
  3. KaKs_Calculator(https://sourceforge.net/projects/kakscalculator2/files/latest/download)计算得出Ka，Ks，ω
  4. 基因对信息，基因CDS和蛋白序列数据
  • 文件类型
  • gene_pair.txt
    • 格式为 gene1 gene2
  • gene_cds.fasta
  • gene_pep.fasta
    注意cds和pep以及gene_pair中的名字保持一致。

输出代码：
perl ParaAT.pl -h gene_pair.txt -n gene_cds.fasta -a gene_pep.fasta -p proc -o output -f axt
cat ./output/*.axt > merge.axt
KaKs_Calculator -i ./merge.axt -o resut.txt
